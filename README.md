Este projeto foi desenvolvido em um ambiente controlado utilizando diferentes sistemas operacionais com foco em segurança ofensiva e testes práticos.

Foram utilizados Kali Linux e BlackArch Linux como plataformas de ataque, devido à ampla disponibilidade de ferramentas voltadas para pentest, e Linux Mint como sistema base para suporte ao ambiente e execução de containers.



![Diretorios](https://raw.githubusercontent.com/Pablo-seixas/Ssimula-o-de-ataque-de-for-a-bruta-com-Medusa-em-ambiente-controlado-Dio/main/diretorios.png)



Este laboratório também inclui a análise de comportamento associado a ransomware em ambiente controlado, demonstrando como arquivos podem ser alterados e como esses padrões podem ser identificados com ferramentas de detecção.

O objetivo foi evidenciar, de forma prática, a geração de artefatos e sua análise dentro de um cenário seguro.
